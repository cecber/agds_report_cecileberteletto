# agds_report_cecileberteletto
	AGDS 2 - report exercises
	
	Land-cover & Spatial- Upscaling:
	Here can the two report exercises be fin for the class AGDS2. 
	Two RMarkdown files, one for each exercise (land-cover classification and spatial upscaling), are in the folders.
		
	Data:
	The data are obtained as mention in the course tutorial.
	Land-cover: https://geco-bern.github.io/handfull_of_pixels/gathering_data.html
	Spatial-upscaling: https://geco-bern.github.io/spatial_upscaling/data.html
	
	Project structure:
	In this project you can find 2 folders, one for each exercises.
	They contain the RMarkdown and their corresponding HTML page
		
	How to reproduce the analysis:
	The code in the RMarkdown files should run on any computer	

	
	Results:
	The results can be seen in the HTML outputed from the RMarkdown.	
		
	
	Dependencies:
	The required R packages are to be fin at the beginning of the 2 RMarkdown.
